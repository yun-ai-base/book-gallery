  - 维度3·知识细节 Top 10 核心概念【原文｜章节+原话/算例摘录】
    - 1) 总纲串联句：「有数据的地方，必有矩阵！有矩阵的地方，更有向量！有向量的地方，就有几何！有几何的地方，皆有空间！有数据的地方，定有统计！」【原文｜第1章】
    - 2) 矩阵乘法=运算核心：「矩阵乘法：线性代数的运算核心」；比奈（Jacques Philippe Marie Binet）1812 年首先提出矩阵乘法规则【原文｜4.5 + 第4章 PDF】
    - 3) 矩阵乘法两个视角：第一视角「标量积展开」、第二视角「外积展开」（分块矩阵章再深化）——乘法是信息重组而非普通数乘【原文｜4.6、6.2/6.3】
    - 4) 广播原则（broadcasting）：矩阵 A + 行向量 r → r 自我复制上下叠加成同形矩阵再相加；列向量 + 行向量亦可通过广播完成【原文｜4.4 + 第4章 PDF 式(18)(19)】
    - 5) 真实算例·内积与柯西-施瓦茨：向量内积=标量积=点积，结果为标量；由 cosθ∈[−1,1] 推出柯西-施瓦茨不等式 a·b ∈ [−‖a‖‖b‖, ‖a‖‖b‖]【原文｜2.6 + 书摘】
    - 6) 真实算例·范数与超椭圆：Lp 范数是 L2 推广；L1=旋转正方形、L2=正圆、L∞=正方形；等 L2 范数向量终点落于同心圆【原文｜第3章 + 2.3 书摘】
    - 7) Python 落地：numpy.linalg.norm() 默认 L2；numpy.dot() 对一维数组算内积、对矩阵等价于 @；numpy.vdot() 先展开成向量再内积【原文｜2.3/2.6 代码段】
    - 8) 真实算例·均值即矩阵运算：E(xj) = (1/n)·1ᵀxj，其中 1 为全 1 列向量——统计均值用线性代数一眼算出【原文｜第22章 PDF 式(3)(4)】
    - 9) 真实算例·特征值分解解斐波那契：第 14 章用方阵对角化求斐波那契数列通项式、求马尔科夫过程平稳状态、瑞利商【原文｜14.3/14.4】
    - 10) SVD 普适性：「奇异值分解：适用于任何实数矩阵」（特征值分解仅适用于方阵）；几何视角同为「旋转→缩放→旋转」【原文｜11.6、15.1】
    - 原话补遗：第 9–10 章「投影视角看回归」「数据正交化」——回归与降维本质是投影到子空间【原文｜9.7、第10章】

- ④ 反常识预警 🚨（从书里长出，带案例）
  - 🚨 反常识1：「乘法」在这里不交换，且有两种读法。案例：矩阵乘法 AB≠BA；书中给「标量积展开 / 外积展开」两个视角——乘法实为信息的重组与投影，而非数的普通相乘；向量×矩阵×向量得到二次型，远超出算术直觉【原文｜4.5/4.6、6.2/6.3】
  - 🚨 反常识2：「距离」不止一种形状。案例：L1 范数 → 旋转正方形，L2 → 正圆，L∞ → 正方形（统称超椭圆）；日常欧氏距离只是 p=2 的特例，机器学习里换范数就等于换「形状偏好」【原文｜第3章】
  - 🚨 反常识3：统计均值、回归、降维是同一个东西——矩阵投影。案例：第 22 章用 E(xj)=(1/n)·1ᵀxj 一句话算出均值；第 9–10 章「投影视角看回归」、数据正交化——看似三门课，本质都是把数据投影到子空间【原文｜第22章、9.7、第10章】
