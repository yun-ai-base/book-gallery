# 《芯片战争》· L3 研究

- 〇 书籍元数据【原文】
  - 作者：余盛（战略咨询出身、半导体产业观察者）
  - 出版社：华中科技大学出版社｜2022（ISBN 978-7-5680-7661-6，定价 59.80 元）；另有 2025 再版（49 元，B5 开本）
  - 载体形态：419 页（16+403），有书目（414–415 页）｜中图分类 TN43-11｜并列名 War of chips
  - 学科定调【待你确认】：主学科=产业经济 / 科技战略；辅助学科=半导体工程、国际政治经济学、商业史
  - 读者定位：半导体从业者、打算投资芯片的投资者、未定职业方向的学子、关心"卡脖子"的普通读者

- ① 前置元评估（止损门）【推导｜基于简介+读者评】
  - 核心创见：把芯片史讲成"企业竞争 + 国力较量"双线叙事，提出"摩尔定律=赢家通吃的指数游戏"，并用华为 2019 被制裁的"楔子"把全书钉在现实焦虑上
  - 判定：✅ 继续（值得研究级拆解）——它是中文世界少有的"芯片全产业链商业史"，L3 价值在于用产业组织与国际政治坐标校准其"中国叙事"倾向
  - 绑定条件：L3 研究 → 维度4 执行全部三层，并补延伸书单与文献对照

- ② 五维深度拆解
  - 维度1·背景 Why【原文｜序+楔子】
    - 写作动机（原话）："讲清楚那些与芯片相关的公司的商业成功逻辑，着实要比讲明白那些深奥的芯片技术知识更加重要"
    - 楔子锚点（原话）："麒麟芯片之所以在 9 月份以后无法生产，是缘于美国商务部当年 5 月 15 日针对华为出台的第二轮出口管制措施……三家垄断芯片 EDA 工具的美国企业不能为海思提供设计工具，台积电和其他制造厂也不能为海思制造芯片"
    - 立论基调（原话）："中华民族要想取得伟大复兴，在很大程度上有赖于一块小小的芯片"
  - 维度2·逻辑架构 What/How【原文｜目录】
    - 楔子 华为的难题
    - 上部 全球芯风云：第一章 从晶体管到芯片（晶体管替代真空管 / 德州仪器和仙童发明芯片 / 军用走向民用）｜第二章 存储器公司（英特尔提出摩尔定律 / 日本 VLSI 计划）｜第三章 英特尔向 CPU 转型（美日芯片战争 / 只有偏执狂才能生存）｜第四章 存储器新势力（美光 / 闪存诞生 / 三星内存崛起 / 台湾内存产业）｜第五章 阿斯麦尔（飞利浦光刻机 / 愚人节成立 / 从超威突破）｜第六章 芯片食物链（工研院院长 / 蔡司镜头传奇）｜第七章 美国完胜海湾战争（日本可以说"不"？/ 韩国内存击败日本 / 苏联被摩尔定律抛弃 / 中国卧薪尝胆）｜第八章 亚洲金融危机（三星神话 / 海力士 / 台湾内存危局）｜第九章 晶圆代工群雄（台湾双雄 / 格罗方德 / 光刻干湿大战）｜第十章 金融风暴大洗牌（尔必达苦恼 / 内存大败局 / 三分天下）｜第十一章 日本芯片的黄昏（东芝售闪存 / 日本电子衰败）
    - 下部 中国芯势力：第十二章 "909 工程"始末（华虹 NEC）｜第十三章 设计中国芯（自主 CPU / 展讯）｜第十四–十七章 中芯国际、张汝京、兆易创新、大陆存储器突破等
    - 【贯穿全书锚点】"摩尔定律的残酷性"：技术指数迭代 → 研发投入指数增长 → 利润回报与资本投入须成正循环，否则"亏损黑洞之大，即便大国也难承受"
  - 维度3·知识细节 Top 10 核心概念【原文｜章节+原话/数据】
    - 1) 摩尔定律即"赢家通吃"（原话）："摩尔定律意味着芯片技术的竞争是相当残酷的，第一名往往能够吃掉行业半数营收和大部分的利润，第二名还能喝到一些汤，再后来者只能吃些残羹冷饭"
    - 2) 正循环铁律（原话）："利润回报与资本投入必须形成正循环，否则在芯片事业上的亏损黑洞之大，即便是一个大国或强国也无法承受"
    - 3) 华为楔子（数据+案例）：2019-05-15 美第二轮出口管制 → 三家美企 EDA 工具断供 + 台积电等制造厂拒接海思 → 麒麟 9 月后停产
    - 4) 台积电代工改写格局（案例+原话）：1998 英伟达把显卡交台积电代工，次年提出 GPU 概念跃居显卡老大；黄仁勋说"如果等我自己建厂生产 GPU，我现在可能就是一个守着几千万美元公司的安逸 CEO"；"没有 Plan B，全部压在台积电上"
    - 5) 张忠谋代工逻辑（原话）："Intel 设计 CPU 很厉害，但是生产 CPU 不厉害，我的成本是他的一半，我的品质比他的好两倍，我可以帮你代工"；预言"二十年后连英特尔的微处理器也将转型代工"
    - 6) 英特尔顶点数据（数据）："1999 年是英特尔的一个辉煌顶点，它的市值在这一年达到 5090 亿美元的历史最高峰"
    - 7) 硅片高度垄断（数据）：全球前 5 大硅片商（信越化学、三菱住友、世创、硅德荣、环球晶圆）控制市场 94%，12 英寸大硅片控制度高达 98%
    - 8) 手机芯片份额（数据，Gartner）：2017 年展锐出货量全球占比 11%，对手高通 36%、联发科 24%；2014 英特尔斥资 15 亿美元收购展锐 20% 股份
    - 9) 兆易创新起步（数据+案例）：朱一明 32 岁创业，开发每单元 2 晶体管 SRAM（市面为 6 晶体管），成本压缩 2/3、性能效率提 3 倍；凑 92 万美元创办
    - 10) 产业产值结构（数据）："全球半导体产业晶圆代工、芯片设计和 IDM 厂三大类型公司的产值比例大致是 1∶2∶3"；芯恩一期二期总投资约 200 亿元，走特色工艺而非先进制程
    - 原话补遗："信息社会盛行的是'赢家通吃'的游戏规则，只有极少数胜利者能够赢得牌桌上几乎所有的筹码"
  - 维度4·批判性逻辑 L3（三层全开）【原文｜书评+外援】
    - 子层A·审稿人视角（本书内部的逻辑漏洞）【推导｜基于书评与局限自陈】
      - 漏洞1·数据多为二手转引：书中 94%/98%、5090 亿、92 万美元等数字来自读书笔记/行业报告转引，未给原始出处页码，存在时点漂移（如英特尔市值、展锐份额随年变化）
      - 漏洞2·"中国叙事"乐观偏差：下部"中国芯势力"以张汝京、赵伟国等人物传奇为主线，对技术代差（EUV 光刻、EDA、IP 核）的结构性困境着墨轻，易给读者"即将反超"的错觉
      - 漏洞3·工程深度不足：作者承认"讲商业逻辑比讲技术知识更重要"，对制程、良率、光刻原理仅点到为止，硬科技读者会觉得"隔靴搔痒"
      - 漏洞4·以企业史代国策史：把"芯片战争"收敛为几家公司的商战，弱化了出口管制、瓦森纳协定、地缘联盟等国家级制度变量
    - 子层B·对立学者视角（其他范式如何反驳本书）【外援｜国际政治/创新学派】
      - 地缘政治经济学（如《芯片战》米勒 Chris Miller）：本书偏"企业商战"，而米勒强调"国家用出口管制当武器"——芯片战争本质是权力战争而非市场战争，本书对"美国如何用规则锁死对手"剖析不足
      - 创新系统学派（伦德瓦尔/ Freeman）：本书崇拜"个人英雄（张忠谋、张汝京）"，忽视国家创新系统（产学研、政府采购、标准）的支撑，易陷入"伟人史观"
      - 依附理论：本书把"台湾内存跟风、中国大陆代工全军覆没"归为市场失败，未触及"全球价值链中半边缘被核心锁定"的结构性依附
    - 子层C·学科史视角（这套知识在学科脉络里的位置与演变）【外援｜产业史/科技史】
      - 1958 基尔比（德州仪器）与 1959 诺伊斯（仙童）发明集成电路，芯片史起点；本书第一章即此
      - 1971 英特尔 4004 微处理器 + 1965 摩尔定律提出，开启"指数迭代"产业逻辑
      - 1980 台积电开创纯晶圆代工（张忠谋），重构全球分工——本书"代工改写格局"的主轴
      - 2018–2019 中美贸易战 + 华为制裁，使"芯片=国力武器"从学术命题变现实，本书楔子正落在此节点，是产业史写作的"当下性"标本
  - 维度5·连接与迁移【推导】
    - → 投资：用"赢家通吃 + 正循环"筛赛道——只有能形成"利润↔资本"正反馈的环节才值得投（如设备/材料垄断端）
    - → 国家战略：楔子证明"EDA+代工"任一断点即可瘫痪整机，启示"自主须全链而非单点"
    - → 个人职业：书中"设计:代工:IDM = 1:2:3"提示产业链价值分布，选岗时可对标高份额环节

- ③ 概念树（以"摩尔定律的残酷性"为根）【原文｜序+第一章+楔子】
  - 根：芯片竞争 = 指数游戏（摩尔定律）
    - 前提：技术每 18–24 月翻倍 → 研发/设备投入须指数增长才能保领先
    - ↓ 推导/演变
      - 正循环成立：利润回报 ↔ 资本投入 正反馈 → 赢家通吃（第一吃半数营收）
      - 正循环断裂：落后 → 亏损黑洞（大国也难承受）→ 出局（苏联被抛弃、台湾内存危局）
      - 分工重构：IDM → 设计/代工/IDM 三分（产值 1:2:3），台积电代工成"卡位点"
    - ↓ 结论/应用
      - 华为楔子：EDA 断供 + 台积电拒接 → 麒麟停产 = "任一卡点即瘫痪"
      - 中国路径：909 工程 / 中芯 / 兆易 / 芯恩（200 亿走特色工艺）= 在成熟制程找突破口

- ④ 反常识预警 🚨（从书里长出，带案例）
  - 🚨 反常识1：芯片不是"越自主越好"，而是"任一环节断供即归零"。案例：华为海思能设计旗舰麒麟，却因美国三家 EDA 工具 + 台积电制造同时断供，2019 年 9 月后直接停产——证明"设计强≠能造出"，全链自主才是真安全【原文｜楔子】
  - 🚨 反常识2：行业老二都"喝汤"，老三只能"吃残羹"，且研发投入却要指数涨。案例：英特尔 1999 年市值冲到 5090 亿美元的顶点，但摩尔定律下后来者（AMD/台积电）靠代工与架构迭代反超，说明"昨天的冠军"也可能被正循环甩出牌桌【原文｜第二/三章】
  - 🚨 反常识3：看似最"低级"的硅片，垄断程度比 CPU 更吓人。案例：全球 5 家硅片商控 94% 市场、12 英寸硅片控 98%，连张汝京都只能先建上海新昇补短板——大众盯着 CPU 大战，真正的"卡脖子"藏在更上游的材料里【原文｜第十六章读书笔记数据】

- ⑤ 学科差异化结构化输出（产业经济/科技战略视角）【推导】
  - 阅读顺序：先读楔子（华为难题，建立痛感）→ 序（摩尔定律残酷性）→ 上部全球史 → 下部中国史，倒着建立"为什么难"
  - 一句话记忆：摩尔定律=指数游戏；利润↔资本正循环=生死线；设计:代工:IDM=1:2:3
  - 三个卡点自查：EDA、光刻机（尤其 EUV）、硅片材料——任一缺口即系统性风险
  - 延伸书单（接住 L3 向外延伸，按用途分层）【外援｜选书建议】
    - 地缘视角补强：Chris Miller《芯片战》（Chip War）——补本书"国家用管制当武器"的短板
    - 技术硬核：摩尔《芯片简史》/ 行业白皮书——补本书"工程深度不足"的盲区
    - 中国路径对照：陈芳/董瑞丰《芯路》或行业报告——看 2019 后真实突破与差距
    - 创新系统：伦德瓦尔《国家创新系统》——把"个人英雄"叙事拉回制度视角

  - 文献对照（三种范式并置，看清本书坐标）【外援｜范式对照】
    - 余盛"企业商战史（商业逻辑优先）" vs 米勒"地缘权力史（管制即武器）"：同一芯片史，前者乐观市场化，后者冷峻权力化
