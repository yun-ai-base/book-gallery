# 《中国哲学十九讲》· L3 研究

- 〇 书籍元数据【原文】
  - 作者：牟宗三（1909—1995，字离中，山东栖霞人；现代新儒学集大成者；思想受熊十力影响极大，谋求儒家哲学与康德哲学融通，重建「道德的形上学」）
  - 出版社与版本：台湾学生书局 1983-10 初版（29.2 万字）；上海古籍出版社 2005（ISBN 9787532539055，371 页）；贵州人民出版社 2020-03（ISBN 9787221158734，392 页，定价 45 元）
  - 豆瓣评分 9.0（基于 799 人评价）【原文｜百度百科引豆瓣】
  - 分类：哲学宗教-东方哲学；关联著作《才性与玄理》《佛性与般若》《心体与性体》《从陆象山到刘蕺山》《圆善论》《名家与荀子》
  - 学术定位：哲学大家刘述先评——「牟宗三在当代中国哲学的地位比之于康德在西方哲学的地位」【原文｜作者简介引刘述先】
  - 读者定位：已具备基本中国哲学/西方哲学常识、想贯通儒释道系统义理的进阶读者

- ① 前置元评估（止损门）【推导｜基于简介+书评】
  - 核心创见：以「生命的学问」为主轴，用康德式概念工具（形而上学、分别说/非分别说）重梳中国哲学从先秦到宋明的系统纲格，提出「通孔说」「两种真理」「纵贯系统」「三系说」等原创判教框架
  - 判定：✅ 继续（值得研究级拆解）——它是牟宗三对中国哲学的「综述」而非入门，L3 价值在看清他用一套西哲概念「创造性重建」中学的功与过，以及由此引发的学派争论
  - 绑定条件：L3 → 维度4 三层全开（审稿人+对立学者+学科史），并补延伸书单与文献对照
  - 风险预警：本书是「判教」而非「客观史」，牟宗三的「三系说」「圆教」带有强烈价值排序，须与反对意见并置

- ② 五维深度拆解
  - 维度1·背景 Why【原文｜序+创作背景】
    - 成书由来：1978 年对台大哲学研究所诸生口讲，本无意成书；是在《才性与玄理》《佛性与般若》《心体与性体》《从陆象山到刘蕺山》基础上对中国各期哲学的「综述」【原文｜xueruxue 引序】
    - 思想动因：回应「中国是否有哲学」的合法性质疑；熊十力→牟宗三一脉谋求「儒家哲学与康德哲学融通」，重建「道德的形上学」【原文｜百度百科创作背景】
    - 定位矛盾：它自称「切关中国哲学之系统纲格与义理宗趣之综述」，实则是高度个人化的「判教」——综述外衣下的立场之书

  - 维度3·知识细节 Top 10 核心概念【原文｜章节+原话摘录】
    - 1) 通孔说（第一讲）：每个文化系统都从一个「通孔」表现自己，中国哲学从尧舜禹汤文武周公这个通孔发展出以生命为对象的课题【原文｜百度百科"通孔说"】
    - 2) 生命的学问：原话「中国哲学，古人重视生命问题，现在没有人重视这个生命问题。现在人把生命首先变成心理学，然后由心理学变成生理学……这一看，把人都看没有了。」【原文｜读书笔记引牟宗三原话】
    - 3) 两种真理（第二讲）：外延真理（科学的、可客观肯断、脱离主体）vs 内容真理（系属于主体生命、德性）；中国哲学所重在内容真理。借罗素对两真理的区分为框架【原文｜百度百科+"缘起性空"文】
    - 4) 周文疲弊（第三讲）：春秋战国诸子皆是对「周朝礼乐制度失效」这一时代病所开药方——这是牟宗三解释诸子起源的核心设定【原文｜今日头条书评】
    - 5) 儒家系统重主体（第四讲）：以五经为据，儒家性格在「重主体」；用康德「形而上学」定义批评两种误读——只讲道德不讲存在、只讲中庸不讲理学【原文｜点藏网】
    - 6) 纵贯横讲 + 作用的表象（第六、七讲）：道家玄理为「境界形态的形上」；道非客观实体创生万物，而是主观上「让开一步、不禁其性」的智慧；「道」之表象是「作用地」而非「存实地」【原文｜点藏网+百度百科】
    - 7) 法家完成君主专政（第八、九讲）：法家自觉解决周文疲弊的急迫问题，其打击贵族、尊君内涵推动「从周制到君主专制」的历史转型【原文｜百度百科】
    - 8) 一心开二门（第十四讲）：借《大乘起信论》「一心开二门」(心真如门/心生灭门) 框架安置真如与现象，为佛教哲学的圆教表达提供结构【原文｜目录+ccbbp】
    - 9) 圆教与圆善（第十五、十七讲）：佛教中圆教的意义；「圆善」= 德福一致之最高圆满，连接康德《实践理性批判》的「至善」问题【原文｜目录】
    - 10) 三系说（第十八讲）：宋明儒学分三系——①周敦颐、张载、程颢→胡宏、刘宗周；②程颐→朱熹（格物穷理、心性二分）；③陆九渊→王阳明（心即理、良知）。这是牟宗三最具争议的判教结论【原文｜百度百科】
    - 金句补遗【原文｜ccbbp】：「中国人一开始关心的就是人，而不是物，不是上帝」「西方哲学是『为实有而奋斗』，佛教则是『为去掉实有而奋斗』」
  - 维度4·批判性逻辑 L3（三层全开）【原文｜书评+外援】
    - 子层A·审稿人视角（本书内部逻辑漏洞）【推导｜基于附录标题"客观的了解与创造性的重建"+学界争议】
      - 漏洞1·以「判教」冒「综述」：声称是系统纲格综述，却用「纵贯系统」「三系说」暗含价值高下，把个人重建当客观史，易误导读者以为这是定论
      - 漏洞2·强以康德框架剪裁中学：用「分别说/非分别说」「形而上学」硬套儒释道，造成「中学西解」的格义风险，某些概念（如圆善）被过度康德化
      - 漏洞3·三系说对朱子的贬抑：将程朱系置于「旁出/歧出」，引发后世（尤其尊朱学者）强烈反弹，其「正宗在陆王」的判断并非无争议
      - 漏洞4·轻忽非正统与民间：聚焦儒释道主流与名家，对墨家（仅提及「以功利态度非难周礼」）、法家之外的制度思想及民间儒学着墨有限
    - 子层B·对立学者视角（其他范式如何反驳本书）【外援｜外部学派】
      - 唯物主义/马克思主义哲学史（如任继愈《中国哲学史》）：拒绝「生命的学问」「纵贯系统」的唯心前提，主张从社会存在/经济基理解诸子起源，与牟的「周文疲弊」心性化解释对立
      - 文献考证派（如劳思光《新编中国哲学史》）：劳思光以「义理的外观」与「基源问题」方法，反对牟宗三以系统建构掩盖历史考证，对宋明儒学分期亦不同
      - 分析哲学/比较哲学（如陈汉生、安乐哲）：质疑「content truth/外延真理」二分是否偷运了西方范畴，主张以「角色/关系」而非「实体/主体」重读中国哲学，反对牟的康德式重构
    - 子层C·学科史视角（这套知识在学科脉络里的位置）【外援｜哲学史】
      - 现代新儒学「三期说」：熊十力(开宗)→牟宗三/唐君毅/徐复观(集大成)→当代（杜维明等），牟是「良知自我坎陷」说与「三系说」的提出者
      - 康德接受史：牟译康德三大批判并以其「现象/物自体」「实践理性」为枢纽重建儒家的「道德的形上学」，是 20 世纪中学与西哲最系统的对接尝试
      - 「判教」传统：佛教天台宗原有「藏通别圆」判教；牟宗三把判教方法移用于儒释道整体排序，是方法论上的创造性（亦受批评）移植
      - 圆教问题源流：从智顗《法华玄义》圆教，到牟宗三以「分别说/非分别说」重释，连接康德「至善/圆善」——体现其「以中解西、以西范中」双向操作
  - 维度5·连接与迁移【推导】
    - → 比较哲学：用「两种真理」框架对照科学哲学（波普尔证伪/库恩范式）对「客观真理」的处理，看清内容真理的边界
    - → 伦理学：三系说直接关联「道德动力从何来」——程朱(性即理/穷理) vs 陆王(心即理/良知)，可接《传习录》与《近思录》对读
    - → 宗教哲学：一心开二门、圆教，是理解佛教哲学化（与中国化）的关键结构，可接《大乘起信论》《法华玄义》
    - → 文化诊断：以「通孔说」「生命学问」反观当代工具理性膨胀，呼应韦伯「祛魅」与泰勒「本真性」议题

- ③ 概念树（以「中国哲学特殊性」为根，长出系统纲格）【原文｜第一~三讲立意】
  - 根：中国哲学的特殊性 = 以生命为对象、向内的「生命的学问」
    - 前提：每种文化自一「通孔」开端（中国通孔=尧舜禹汤文武周公）→ 推导：中国哲学重内容真理、重主体生命
    - ↓ 沿时间轴展开系统
      - 枝·先秦：周文疲弊 → 儒(仁/重主体)、道(作用的表象/纵贯横讲)、法(君主专政)、名(形名)
      - 枝·魏晋隋唐：玄学(无为) + 佛教(缘起性空→一心开二门→圆教)
      - 枝·宋明：三系说（周张程颢胡宏刘宗周 / 程颐朱熹 / 陆王）→ 结论：纵贯系统的圆熟 = 天人合一完成
    - ↓ 勾稽：以「纵贯系统」贯穿儒释道（儒家最彻底，道佛纵贯横讲），以「分别说/非分别说」表达圆教
      - 结论：中国哲学的系统纲格 = 内容真理 + 纵贯系统 + 圆善（德福一致）

- ④ 反常识预警 🚨（从书里长出，带案例）
  - 🚨 反常识1：中国哲学「不讲逻辑、不重外在」，却被牟宗三判为「最彻底的纵贯系统」。案例：第一/四讲——西方以「为实有而奋斗」的实体思维衡量中学，会误以为中国无哲学；牟反以「内容真理」证明中学另有一套更根本的普遍性【原文｜两种真理+纵贯系统】
  - 🚨 反常识2：法家「解决问题」的结果是把中国推向君主专制。案例：第八/九讲——法家自觉医治「周文疲弊」，看似高效，却完成「打击贵族、尊君」的政治格局转型，与儒道「安顿生命」方向背道而驰【原文｜法家两讲】
  - 🚨 反常识3：宋明儒学的「正统」不是朱熹而是陆王。案例：第十八讲三系说——牟宗三把程颐→朱熹系判为「旁出/歧出」，以陆九渊→王阳明「心即理」为纵贯正宗，直接颠覆通行教科书叙事【原文｜三系说+外援｜尊朱学者反弹】

- ⑤ 学科差异化结构化输出（东方哲学/新儒学视角）【推导】
  - 阅读顺序：先啃第一~三讲（特殊性/两种真理/诸子起源）立骨架 → 再按兴趣进儒(4)、道(5-7)、法(8-9)、佛(12-17)、宋明(18-19)
  - 一张表记住：通孔→生命学问；外延真理(科学) vs 内容真理(生命)；纵贯系统(儒最彻底)；三系(周张程颢/程朱/陆王)
  - 避坑：把「牟宗三的判教」与「中国哲学史实」分开——本书是后者的一家之说
  - 延伸书单（接住 L3 向外延伸）【外援｜选书建议】
    - 牟宗三自著进阶：《心体与性体》——三系说的完整论证源头；《圆善论》——圆善问题的专书
    - 对照哲学史：劳思光《新编中国哲学史》——以「基源问题」法提供不同于牟的分期与判准
    - 唯物范式：任继愈《中国哲学史》——社会存在视角的对立叙事
    - 比较哲学：安乐哲《通过孔子而思》——以「角色/关系」反牟的实体/主体式重构
    - 康德原典：康德《实践理性批判》——圆善/至善问题的西源，理解牟「良知自我坎陷」的对接点
  - 文献对照（三种范式并置）【外援｜范式对照】
    - 牟宗三「判教式综述(纵贯系统/三系)」 vs 劳思光「基源问题式哲学史」：取舍在「系统建构」vs「历史考证」
    - 牟「内容真理/生命学问」 vs 科学哲学「外延真理/客观知识」：中学特殊性的自我辩护，也是与实证主义的根本分歧
    - 牟「以康德范中学(圆善/一心开二门)」 vs 分析/比较派「去康德化重读」：20 世纪中学西释的最尖锐争论点
