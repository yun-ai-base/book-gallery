# 《战国史》· L3 研究

- 〇 书籍元数据【原文】
  - 作者：杨宽（先秦史大家，初版 1955，积五十年治学研究；1998 增订为本，2003 上海人民出版社终版）
  - 出版社：上海人民出版社｜2003 年 4 月第 1 版｜精装｜中国断代史系列
  - 页数 738 页｜字数 56 万｜定价 ¥51｜ISBN 9787208045378｜豆瓣 8.8（417 人评价）
  - 学科定调【待你确认】：主学科=中国古代史/战国断代史；辅助学科=考古学、经济史、思想史
  - 读者定位：历史专业研究者与爱好者、制度史/经济史入门者；号称"读这一本就够了"的战国通史

- ① 前置元评估（止损门）【推导｜基于简介+书评】
  - 核心创见：学界第一部用马克思主义史观系统写就的战国通史；以"从分裂割据走向统一"为轴，把经济（铁器/水利/小农）、制度（郡县/军功爵）、战争、百家争鸣打通；并注重用考古新史料补充、纠正传世文献，以历史地图作空间视角
  - 判定：✅ 继续（值得研究级拆解）——它是战国断代史的"扛鼎之作"与集大成者，L3 价值在于把它作为制度史锚点，向外接入其他史学范式与考古新发现
  - 绑定条件：L3 研究 → 维度4 执行全部三层（审稿人+对立学者+学科史），并补延伸书单与文献对照

- ② 五维深度拆解
  - 维度1·背景 Why【原文｜前言+初版自陈】
    - 写作动机：杨宽要写一部"从分裂割据走向统一，在政治、经济、军事、文化等领域发生的重大变革和发展"的战国全史；自承战国史料"缺乏、年代紊乱、真实性不足、未系统整理"，故极重考古新史料
    - 时代语境：清代王夫之《读通鉴论》称战国为"古今一大变革之会"——杨宽在前言直接采信并展开
    - 关键背景原话："这时农业生产，由于铁工具的普遍使用，水利灌溉工程的开发，生产技术的进步，荒地的开垦，一年两熟制的推行，农田产量很有增加，使得五口到八口之家的小农得以成长"（前言）
  - 维度2·逻辑架构 What/How【原文｜目录】
    - 前言
    - 第一章 绪论（战国史的重要性；战国史料的整理和考订）
    - 第二章 春秋战国间农业生产的发展（冶铁、水利、作物、生产技术、农本理论）
    - 第三章 春秋战国间手工业和商品经济的发展（技术、城市、货币、高利贷、贸易理论）
    - 第四章 春秋战国间社会经济制度的变革
    - 第五章 战国前期各诸侯国的变法改革
    - 第六章 中央集权的政治体制及其重要制度
    - 第七章 七强并立的形势和战争规模的扩大
    - 第八章 合纵、连横和兼并战争的变化
    - 第九章 秦的统一
    - 第十章 战国时代的"百家争鸣"
    - 第十一章 战国时代科学和科学思想的发展
    - 第十二章 战国时代文化的发展
    - 附录一 战国郡表 / 附录二 战国封君表 / 附录三 战国大事年表
    - 【贯穿全书锚点】"从分裂到统一"主线 + "小农经济→中央集权→军功爵→兼并统一"因果链，全书各章均围绕此链展开
  - 维度3·知识细节 Top 10 核心概念【原文｜章节+原话摘录】
    - 1) "古今一大变革之会"（王夫之语，杨宽引为全书定调）：战国非乱世，而是制度文明的转折点【原文｜前言】
    - 2) 铁器革命：铁工具普遍使用 + 水利开发 + 一年两熟制 → 农田增产，小农（五口到八口之家）得以成长【原文｜第二章/前言】
    - 3) 授田制："魏、秦等国先后推行按户籍'良民'身分授田的制度，规定一夫授田百亩"——国家规模的自耕小农成为君主政权立国基础【原文｜前言】
    - 4) 中央集权："废弃了原来由各级贵族统治的制度，开始形成以将相为首脑的中央集权的君主政权，普遍地推行着郡县两级的地方行政组织"【原文｜前言】
    - 5) 商鞅军功爵（硬数字）："秦国的士兵只要斩获敌人'甲士'一个首级，就可以获得一级爵位（公士）、田一顷、宅一处和仆人一个"；斩两首级可释囚犯父母、妻奴隶转平民；五大夫可衣食 300 户租税，杰出者 600 户【原文｜第三章/商鞅变法考证】
    - 6) 二十等爵序列（真实层级）：公士→上造→簪袅→不更→大夫→…→五大夫→…→大庶长→关内侯→彻侯（最高），以敌首数量授爵【原文｜第五章/制度考】
    - 7) 高利贷横行（硬数字）："孟尝君田文，就是一个大高利贷者，他曾在封邑薛地大放高利贷，一次就收到'息钱十万'"（《史记·孟尝君列传》引）；秦律：借债不还服役"每服役一天折合八钱"【原文｜第三章】
    - 8) 券的普遍应用："大市以质，小市以剂"——大买卖用长券"质"，小买卖用短券"剂"（《周礼·质人》），买卖纠纷"听买卖以质剂"【原文｜第三章】
    - 9) 百家争鸣：第十章专论，儒墨道法名阴阳等学派在"古今一大变革"中并起，思想史的制度土壤即前章经济政治变革【原文｜第十章】
    - 10) 史料考订自觉：杨宽点明战国史料"缺乏、年代紊乱、真实性不足"，本书以考古标本作"补充、纠正原有史料并用作鉴别真伪的标本"【原文｜第一章/初版自陈】
  - 维度4·批判性逻辑 L3（三层全开）【原文｜书评+外援】
    - 子层A·审稿人视角（本书内部的逻辑漏洞）【推导｜基于书评与局限自陈】
      - 漏洞1·制度史偏重、生活史偏轻：全书以政治经济制度为骨架，对底层个体生命、日常物质生活、民众心理着墨有限，读来"宏大"却少"体温"
      - 漏洞2·中原中心视角：七强（魏楚燕齐赵韩秦）为主轴，对巴蜀、荆楚边缘、周边少数民族处理简略，空间上不均衡
      - 漏洞3·马克思主义框架的化约风险：以"生产力→生产关系"解释一切变革，对文化、思想、信仰等非经济动因的解释有时显得单线因果
      - 漏洞4·史料依赖传世文献：虽重考古，但战国地下材料（简牍）在 1998 终版时远未如今天（如清华简、里耶秦简）丰富，部分结论已被新出简牍修正
    - 子层B·对立学者视角（其他范式如何反驳本书）【外援｜外部学派】
      - 文化精神史范式（钱穆《国史大纲》）：钱穆重"民族精神与文化命脉"的褒贬叙述，批评唯物史观把历史"经济决定"化；对战国更强调"士"的精神自觉而非制度强制
      - 当代社会史/新史学（如杜正胜"新社会史"、大陆社会史学派）：强调基层社会、家族、日常、物质文化，正补本书"制度自上而下"之盲
      - 考古学独立派（如李学勤"走出疑古"）：更彻底倚重出土文献与器物，对杨宽承顾颉刚"疑古"整理传世史料的方法持修正立场
    - 子层C·学科史视角（这套知识在学科脉络里的位置与演变）【外援｜学科史】
      - 战国叙述之源：司马迁《史记》（战国策素材）开创纪传；此后《资治通鉴》编年、《通鉴纪事本末》为本末体
      - 现代战国史之成立：20 世纪吕思勉、顾颉刚"古史辨"疑古 + 王国维"二重证据法"（文献+考古）奠定范式；杨宽是这一脉络的集大成者
      - 1949 后马克思主义史学成为主流，杨宽本书即该范式在断代史的代表作；1990 后"走出疑古"与考古简牍时代开启新一轮重写
  - 维度5·连接与迁移【推导】
    - → 制度设计：军功爵"以可计量功绩换权益"是早期"绩效制"雏形，可对照现代 KPI/晋升制度
    - → 经济：授田百亩 + 重农抑商 + 高利贷，是"国家榨取型小农经济"的典型结构，可迁移分析任何"国家—小生产者"关系
    - → 跨界：从"铁器+水利→增产→小农→税基→集权"的因果链，可套用于"技术变革如何重塑国家能力"的普遍命题

- ③ 概念树（以"古今一大变革之会"为根）【原文｜前言+各章】
  - 根：战国是"古今一大变革之会"（王夫之语，杨宽引）
    - 前提：铁工具普遍 + 水利开发 + 一年两熟 + 荒地开垦 → 农业增产
      - ↓ 派生
      - 小农成长：五口到八口之家成立；国家"一夫授田百亩"授田制
      - ↓ 派生（立国基础转移）
      - 小农成为君主政权立国基础 → 废弃贵族分治 → 中央集权 + 郡县两级
      - ↓ 派生（激励机制）
      - 商鞅军功爵：斩甲士一首级=公士+田一顷+宅一处+仆人一个（二十等爵）
      - ↓ 派生（结果）
      - 七强并立、战争规模扩大 → 合纵连横 → 秦凭军功爵成"虎狼之国" → 秦统一
    - 并行枝：经济变革（手工业/城市/货币/高利贷）→ 百家争鸣的思想土壤

- ④ 反常识预警 🚨（从书里长出，带案例）
  - 🚨 反常识1：技术大进步的时代，底层农民反而更惨。案例：铁器+水利使农田增产，但农民遇荒年"又称贷而益之"，被迫"抛尸露骨于山沟"（《孟子·滕文公》引）；孟尝君一次放高利贷就收"息钱十万"——进步的红利被租税与高利贷吞噬，与"生产力发展必惠及全民"的直觉相反【原文｜第三章】
  - 🚨 反常识2："战争机器"是靠发家致富激励驱动的，不是靠爱国口号。案例：秦军士兵斩敌甲士一首级即得田一顷、宅一处、仆人一个，斩两首可释父母、解放妻奴；五人为伍连坐，一人逃全伍受罚——把"杀敌=私人收益"制度化，才造出"虎狼之国"【原文｜第五章/第三章考证】
  - 🚨 反常识3：中央集权的"先进"伴随着对个体的彻底量化与监控。案例：授田按户籍"良民"身份、军功按敌首计数、连坐以伍为单位——现代视为"绩效"的其实是早期国家把人变成可计量单位，自由与奴役同源【推导｜基于第五章+第三章】

- ⑤ 学科差异化结构化输出（中国古代史/断代史视角）【推导】
  - 阅读顺序：先读前言（王夫之定调）→ 第二章农业、第三章经济（理解变革土壤）→ 第五章变法、第六章制度（理解机制）→ 第九章统一 → 第十章百家争鸣
  - 一张表记住因果链：铁器水利→小农授田→税基→中央集权郡县→军功爵→兼并统一
  - 读史三件套：① 辨史料（杨宽自陈"年代紊乱"）② 看空间（郡表/封君表/历史地图）③ 连因果（经济→制度→思想）
  - 延伸书单（接住 L3 向外延伸，按用途分层）【外援｜选书建议】
    - 文化精神对照：钱穆《国史大纲》——补本书唯物史观之偏，重民族精神与"士"的自觉
    - 通史同源：吕思勉《先秦史》——同为现代战国史奠基，可与杨宽对读看范式差异
    - 考古新证：李学勤《东周与秦代文明》——"走出疑古"派，以出土文物重述，修正部分结论
    - 制度专深：林剑鸣《秦史稿》——聚焦秦如何从西向东统一，深描军功爵与法家制度
    - 原典回流：钱锺书/诸子《战国策》《孟子》——回到杨宽所引的原始文献现场
  - 文献对照（三种范式并置，看清本书坐标）【外援｜范式对照】
    - 杨宽"马克思主义制度史（生产力决定）" vs 钱穆"文化精神史（民族精神褒贬）"：前者重结构因果，后者重价值评判
    - 杨宽承"古史辨疑古 + 二重证据" vs 李学勤"走出疑古"：对传世史料可信度的态度截然相反
    - 传统纪传/编年（司马迁→司马光） vs 现代断代通史（杨宽）：叙述单元从"人物/帝王"转为"社会结构演变"

- ⑥ 跨界启发与落地验证【推导+外援】
  - 行动清单（5 条）：
    - 1. 读任何断代史先问"史料基础"——像杨宽那样先查"是否年代紊乱、真伪可辨"【原文】
    - 2. 用"铁器→小农→集权"因果链模型，去分析任一"技术变革重塑国家能力"的案例【推导】
    - 3. 设计激励时，记住军功爵逻辑：把组织目标拆成"可计量、可交易"的个人收益单元【原文】
    - 4. 看到"进步时代"叙事，主动找底层被剥削的证据（如高利贷"息钱十万"），避免单线乐观【原文】
    - 5. 用空间视角（郡表/地图）读历史，避免只沿时间轴线性理解【原文】
  - 验证复盘设问（量化指标）：
    - 能否不看目录，默写出"铁器水利→小农授田→中央集权→军功爵→秦统一"因果链？→ 能=真懂主线
    - 拿到一段战国制度描述，能否指出它对应二十等爵的哪一级、靠什么数字授爵？→ 能=内化军功爵
    - 解释"为何技术进步的战国农民仍抛尸山沟"时，能否用"租税+高利贷吞噬红利"说清？→ 能=接住 L3 反常识
