# 《潜规则：中国历史中的真实游戏》· L3 研究

- 〇 书籍元数据【原文】
  - 作者：吴思（1957— ），曾任《农民日报》《中国农民报》记者，后任《炎黄春秋》总编辑；2004 年获《新周刊》"年度知道分子"；另著有《血酬定律》
  - 出版社：复旦大学出版社｜2009 修订版（ISBN 9787309063660，272 页）；初版 2001 云南人民出版社
  - 豆瓣评分约 8.3｜历史-中国古代｜历史社会学/制度史
  - 学科定调【待你确认】：主学科=历史社会学/制度史；辅助学科=新制度经济学、博弈论、政治社会学
  - 读者定位：历史爱好者、公职人员、对社会运行非正式规则有洞察需求的读者
  - 副标题线索："中国历史中的真实游戏"——揭示正式制度之下实际支配社会运行的不成文规矩，作者自称"姑且称之为潜规则"

- ① 前置元评估（止损门）【推导｜基于简介+书评】
  - 核心创见：创制"潜规则"一词，用"合法伤害权""第二等公平""崇祯死弯"等工具，把明清官场中"宣称的原则"与"真实的利害计算"之间的断裂系统呈现
  - 判定：✅ 继续（值得研究级拆解）——它把抽象制度分析落到具体案例，L3 价值在于借"潜规则"框架向外接入新制度经济学、黄仁宇/秦晖等史学对照
  - 绑定条件：L3 研究 → 维度4 执行全部三层（审稿人+对立学者+学科史），并补延伸书单与文献对照
  - 敏感度判定：本书为公开出版的历史学研究，剖析明清官场非正式规则，属历史题材，未触及劳改营/古拉格/政治运动/敏感政治人物，依规不触发跳过

- ② 五维深度拆解
  - 维度1·背景 Why【原文｜自序+修订版序】
    - 写作动机：作者自陈"在仔细揣摩了一些历史人物和事件之后，我发现支配这个集团行为的东西，经常与他们宣称遵循的那些原则相去甚远……真正支配这个集团行为的东西，在更大程度上是非常现实的利害计算"，故命名"潜规则"
    - 实证触发：作者 1980 年代当记者调查开封"平价化肥被层层批条转手"——每一层都在往外批条，"大官批多少、小官批多少，都有一套默契的规矩"，正是"价格双轨制"下权力牟利
    - 定位矛盾：既是"亦雅亦俗、亦庄亦谐"的历史通俗读物，又因把"仁义道德/清正廉明"还原为利害计算而令人"心生寒意"
  - 维度2·逻辑架构 What/How【原文｜目录+原话】
    - 自序：关于"潜规则"和这本书
    - 正编（按三类关系展开）：
      - 官吏 vs 百姓：《身怀利器》《老百姓是个冤大头》《第二等公平》
      - 官吏 vs 上级/皇上：《当贪官的理由》《恶政是一面筛子》《皇上也是冤大头》
      - 官场内部：《摆平违规者》《论资排辈也是好东西》
      - 混合：《新官堕落定律》《正义的边界总要老》《官场传统的心传》《晏氏转型》
      - 总结：《崇祯死弯——帝国潜规则的一个总结》《有关潜规则的定义》
    - 杂编：笑话天道 / 古今中外的假货 / 我们的人格理想 / 理解迷信
    - 代后记：农民与帝国；附录：十周年专访 + 洋人的"权利"我们的"分"（提要）
    - 【贯穿全书锚点·潜规则五条定义】：①私下认可的行为约束；②依当事方造福/加害能力自发生成、降低交易成本；③越界必遭报复、强化预期稳定；④背离正义或正式制度、故以隐蔽形式存在且当事人明确认可；⑤借隐蔽把正式规则代表屏蔽或拉入私下交易以谋利【原文｜《有关潜规则的定义》】
  - 维度3·知识细节 Top 10 核心概念【原文｜篇目+原话摘录】
    - 1) 合法伤害权（身怀利器）：官吏"身怀利器，杀心自起"——掌握低成本合法伤害能力，可凭小借口"瞒天过海"榨取百姓【原文｜《身怀利器》】
    - 2) 第二等公平：第一等=市场等价交换/合理明文法；第二等=官府强加却广被接受的公平（如直接征收草料不付钱）；第三等=衙役额外勒索费【原文｜《第二等公平》+书评案例：清驿站征草料】
    - 3) 恶政是一面筛子：恶政淘汰清官、培育恶棍——"帝国制度很善于把常人难免的弱点和毛病培育为全国性的灾难"【原文｜《恶政是一面筛子》金句】
    - 4) 皇上也是冤大头：信息经各级官吏层层加工，"说真说假被各级掌控"，最高权力者反而拿不到准确信息【原文｜《皇上也是冤大头》】
    - 5) 新官堕落定律：朱元璋发现新官初任"既忠诚又坚持原则"，久任则"又奸又贪"——学红头文件，到实践全不是一回事【原文｜《新官堕落定律》案例：朱元璋】
    - 6) 正义的边界总要老：评判标准随时间漂移——"1952 年贪污十万死刑，2000 年左右贪污百万才死刑，贪污十万坐几年牢"；道德边界亦然【原文｜《正义的边界总要老》数字案例】
    - 7) 晏氏转型：晏子治东阿，前期不媚上不欺下反被责"搞乱了"，后期欺下媚上反被赞"有功"——合乎仁义的政策被迫向潜规则转变的制度变迁模型【原文｜《晏氏转型》案例：晏子春秋】
    - 8) 崇祯死弯（U 形谷底）：多征粮饷本欲练军平乱，潜规则下却"官员贪污+地主压榨→朝廷拿不到钱、百姓投闯王"；压迫越重反贼越多，如 U 形谷底勒死明廷【原文｜《崇祯死弯》+书评推演】
    - 9) 论资排辈也是好东西：官场内部对潜规则心领神会，论资排辈反而降低冲突交易成本【原文｜《论资排辈也是好东西》】
    - 10) 官场传统的心传：潜规则靠"心传"代际延续，破规者受罚，故青天成民族梦【原文｜《官场传统的心传》】
    - 原话补遗：「恶政选择了恶棍」——清官遭淘汰、恶棍被选中，是潜规则发挥效应的结果【原文｜金句】
  - 维度4·批判性逻辑 L3（三层全开）【原文｜书评+外援】
    - 子层A·审稿人视角（本书内部的逻辑漏洞）【推导｜基于书评与局限自陈】
      - 漏洞1·还原论风险：以"现实的利害计算"解释一切官场行为，可能遮蔽了儒家伦理、乡约、个人信念对官员的真实约束
      - 漏洞2·概念边界模糊：作者自认"找不到合适名词"才称潜规则，五条定义偏理想型，实践中与"正式制度""习俗"难以一刀切
      - 漏洞3·二元对立简化：把"正式规则"与"潜规则"对立，弱化了正式制度自身的弹性与官员在其中的裁量空间
      - 漏洞4·现代延伸薄弱：末了只说"加强民主法制才能根除"，对当代复杂社会的潜规则生成/萎缩机制论证不足
    - 子层B·对立学者视角（其他范式如何反驳本书）【外援｜外部学派】
      - 黄仁宇：以"数目字管理"缺位解释明亡，《潜规则》自认"吸收补足黄仁宇"，但黄更重财政技术结构，吴思更重微观利害博弈
      - 秦晖：以"黄宗羲定律"（赋税层层加码）与"仰不足以事父母"的农民史观解释王朝循环，补充了吴思偏"官场内部"的视角
      - 新制度经济学（诺斯）：正式/非正式制度常互补共生而非对立，潜规则可视为"非正式制度"，未必全是负面
      - 社会学新制度主义：强调制度的社会建构与合法性机制，对"纯利害计算"假设提出修正
    - 子层C·学科史视角（这套知识在学科脉络里的位置与演变）【外援｜史学史】
      - 传统：二十四史"本纪/列传"叙事 + 儒家褒贬史观，重"忠奸"道德解释
      - 现代转型：傅衣凌"乡族/基层社会"、萧公权《中国乡村》等社会史/经济史，把眼光从朝廷下移到基层
      - 吴思节点：用"潜规则/血酬定律"做跨学科（史学+制度经济学+博弈论）的综合，使"潜规则"一词进入日常汉语
      - 后续：与"新制度经济学""历史社会学"对话，开启"用博弈论重读中国史"的一支通俗学术写作
  - 维度5·连接与迁移【推导】
    - → 组织管理：任何科层都可能在"明规则"下长出"潜规则"，关键看激励与信息结构
    - → 个人职场：识别"第二等公平/第三等公平"的隐性成本，比抱怨规则更重要
    - → 制度设计：降低"合法伤害权"的低成本伤害能力，是遏制潜规则的根本

- ③ 概念树（以"潜规则"为根）【原文｜自序+定义篇】
  - 根：潜规则 = 正式制度下实际支配运行的隐蔽规矩（五条定义）
    - 成因枝：现实的利害计算 + 趋利避害 → 交易各方自发生成约束
    - 表现枝（三类关系）：
      - 官→民：合法伤害权、第二/第三等公平、老百姓是冤大头
      - 官→上：当贪官的理由、恶政筛子、皇上也是冤大头
      - 官→官：摆平违规者、论资排辈、官场心传
    - 演化枝：新官堕落定律、正义边界会老、晏氏转型（制度变迁）
    - ↓ 总收：崇祯死弯（U 形谷底）——帝国在潜规则下被自身压迫勒死

- ④ 反常识预警 🚨（从书里长出，带案例）
  - 🚨 反常识1：皇帝才是"冤大头"。案例：信息经省→府→县层层加工，说真说假、强调隐瞒都被官吏掌控，最高权力者反而拿不到准确情报，被自己的官僚体系屏蔽【原文｜《皇上也是冤大头》】
  - 🚨 反常识2：清官被淘汰、恶棍被选中，不是偶然而是"制度筛选"。案例："恶政是一面筛子"——恶政淘汰清官、培育恶棍；朱元璋叹新官初任廉洁、久任皆贪（新官堕落定律），说明环境而非个人道德决定结局【原文｜《恶政是一面筛子》《新官堕落定律》】
  - 🚨 反常识3：加税越多反而亡国越快——压迫存在"死亡谷底"。案例：崇祯为平李自成加征粮饷，潜规则下却逼反更多百姓、官兵反得补给，U 形"崇祯死弯"把明廷勒死；压迫撞到谷底会产生反向效果【原文｜《崇祯死弯》】
