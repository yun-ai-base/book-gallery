# 《云彩收集者手册》· L3 研究

- 〇 书籍元数据【原文】
  - 作者：加文·普雷特-平尼（Gavin Pretor-Pinney），英国作家；毕业于牛津大学与中央圣马丁艺术与设计学院；2004 年创立「赏云协会」（Cloud Appreciation Society）；所著《观浪者指南》（2010）获皇家学会温顿科学图书奖；另著有《宇宙的答案云知道》（2006）、《尤克里里手册》（2013）
  - 译者：王燕平、张超（均为北京师范大学天体物理硕士；王供职北京天文馆，张供职国家天文台《中国国家天文》）
  - 原作名：*The Cloud Collector's Handbook*｜出版社：译林出版社｜丛书：「天际线」｜2018 年 3 月精装，111 页，定价 45 元，ISBN 9787544771344
  - 序言：北京天文馆馆长朱进作序推荐｜荣誉：豆瓣 2018 年度科学新知好书
  - 供图：赏云协会官方科普读物，105 位来自世界各地的赏云达人供图，收录百余幅摄影作品
  - 读者定位：无任何自然观察经验的新手 & 资深云彩爱好者；可当博物入门指南，也可当观云竞赛计分册

- ① 前置元评估（止损门）【推导｜基于简介+读者评】
  - 核心创见：把「看云」从消遣升级为一门可计分、可分类、可竞赛的「收集」活动——用一套仿生物分类的命名体系 + 趣味记分系统，让大众系统性地识别 46 种云彩与大气光学现象
  - 判定：✅ 继续（值得研究级拆解）——它是中文世界最权威、最易上手的观云工具书，L3 价值在于以它为锚点，向外接入气象学正典、云分类史与光学原理
  - 绑定条件：L3 研究 → 维度4 执行全部三层（审稿人+对立学者+学科史），并补延伸书单与文献对照
  - 重要限定：本书是「手册/图鉴」而非系统气象教材——它教你识别与命名，不深入物理机制；这是它的功能边界，也是 L3 必须补强的缺口

- ② 五维深度拆解
  - 维度1·背景 Why【原文｜创作者背景+序言】
    - 写作动机：云太常见反而被「视而不见」；作者相信「如果更多的人没有目标地抬头看天空，看心理医生的花费就会锐减」——观云让无所事事变得合理
    - 协会渊源：作者 2004 年创赏云协会，本书是协会官方出版物；协会多年游说推动「糙面云（asperitas）」被世界气象组织正式录入《世界云图》——手册是这场民间科学运动的结晶
    - 定位矛盾：既是严谨的科普（分类学源自世界气象组织《国际云图》），又是浪漫的「收集游戏」——科学性与趣味性并存
  - 维度2·逻辑架构 What/How【原文｜目录】
    - 如何收集云彩（含趣味记分系统说明）
    - 云的分类（仿生物分类：族→属→种三阶，外加变种、附属云与附属特征）
    - 十种主要云彩类型（按高度与外观的「属」级）：积云 Cumulus｜层积云 Stratocumulus｜层云 Stratus｜高积云 Altocumulus｜高层云 Altostratus｜卷云 Cirrus｜卷积云 Cirrocumulus｜卷层云 Cirrostratus｜雨层云 Nimbostratus｜积雨云 Cumulonimbus
    - 云种和变种：荚状云 Lenticularis｜毛状云 Fibratus｜堡状云 Castellanus｜波状云 Undulatus｜网状云 Lacunosus｜辐辏状云 Radiatus｜复云 Duplicatus
    - 附属云和附属特征：幞状云 Pileus｜缟状云 Velum｜破片云 Pannus｜悬球状云 Mamma｜幡状云 Virga｜弧状云 Arcus｜管状云 Tuba｜砧状云 Incus
    - 其他的云：山帽云和旗云｜开尔文—亥姆霍兹波｜航迹云｜火积云｜滚轴云｜雨幡洞云｜马蹄涡｜雾和霭｜钻石尘｜贝母云｜夜光云
    - 云的光学现象：虹彩云 Iridescence｜华 Corona｜曙暮光条｜宝光 Glory｜彩虹｜云虹｜22 度晕｜幻日｜环天顶弧｜日柱
    - 专有名词 / 摄影师索引 / 云彩图片索引 / 索引
    - 【贯穿全书锚点】一条分类逻辑：先按高度分「低云族（云底约 2000 米）/ 中云族（2000–7000 米）/ 高云族（约 5000 米以上）/ 多个云族」，再按外观用拉丁词根区分属
  - 维度3·知识细节 Top 10 核心概念【原文｜章节+作者论点】
    - 1) 拉丁词根命名法（书的核心记忆锚）：层 strato=水平分层光滑；积 cumulo=堆积蓬松；卷 cirro=羽状小束；雨 nimbo=蕴藏降水——四词根组合即给出云的特征
    - 2) 十属高度分层：低云（积云/层积云/层云）≈2000 米下；中云（高积云/高层云）2000–7000 米；高云（卷云/卷积云/卷层云）≈5000 米以上；雨层云、积雨云跨层
    - 3) 积云（Cumulus）：最易收集的云，棉絮状，总在晴朗日子慢悠悠飘过——新手第一目标
    - 4) 积雨云（Cumulonimbus）：最具气势、带来雷暴的「多个云族」，顶部常成砧状云（Incus）
    - 5) 分类三阶：仿动植物分类，族→属→种，再加变种与附属云——「云的种类划分有些像动植物的分类法」
    - 6) 趣味记分系统：每种云有对应分值，读者可与亲友展开「观云竞赛」，把识别变成游戏
    - 7) 糙面云（Asperitas）：本书推动被世界气象组织正式命名录入《世界云图》的「新云」，民间科学影响体制标准的范例
    - 8) 云的光学现象群：虹彩云（俗称「七彩祥云」）、华、宝光、22 度晕、幻日、环天顶弧——由阳光经云滴散射/折射/衍射形成
    - 9) 历史坐标：1803 年英国科学家卢克·霍华德（Luke Howard）发表完整云种分类报告，是现代云分类之父
    - 10) 作者原话：「云就像天空的表情，并不是一件可有可无的摆设」——把云读成天空的情绪语言
    - 原话补遗：云由悬浮空中的小水滴或冰晶集合而成（物理本质）；「如果一个人无趣到不会抬头看天……那就不值得跟他一起躺在草地上」——浪漫主义者对观云的态度
  - 维度4·批判性逻辑 L3（三层全开）【原文｜书评+外援】
    - 子层A·审稿人视角（本书内部的逻辑漏洞）【推导｜基于手册体局限与书评】
      - 漏洞1·只教识别不教机制：手册给出名称、特征、分值，但几乎不解释「为何积云蓬松、卷云羽状」的物理成因（上升气流、冰晶形态），读者易停留在「贴标签」
      - 漏洞2·以「收集/计分」游戏化框架包装，可能弱化科学严肃性：为趣味牺牲了天气系统（锋面、气旋）的整体叙事，单朵云脱离了大气环流语境
      - 漏洞3·地域与季节偏差：供图多来自全球赏云达人，部分稀有云（夜光云、贝母云、火积云）在中纬度日常难见，新手计分系统对中国南方/北方常见云的覆盖不均
      - 漏洞4·光学现象部分偏图鉴：华、宝光、22 度晕的原理仅点到为止，未给可操作的观测几何（太阳高度角、云的位置）指引
    - 子层B·对立学者视角（其他范式如何反驳/补充本书）【外援｜外部学科】
      - 气象学教科书范式（如《大气物理学》盛裴轩等）：以动力气象与热力学为骨架，云只是「凝结产物」；本书的「收集视角」在教科书里根本不是主线，机制深度远超手册
      - 天气预报业务范式：关注云系演变（卫星云图、雷达回波）而非「命名收藏」；对业务预报员而言，单朵云的识别价值远低于云系尺度分析
      - 摄影/艺术范式：将云视为光影与构图对象（如风光摄影），不关心拉丁学名——与本书「科学命名优先」取向互补而非冲突
    - 子层C·学科史视角（这套知识在学科脉络里的位置与演变）【外援｜科学史】
      - 1803 年卢克·霍华德在《论云的改造》中确立 stratus/cumulus/cirrus/nimbus 四基型，是现代云分类起点；诗人歌德曾为之写诗
      - 19 世纪《国际云图》（International Cloud Atlas）由世界气象组织前身逐步标准化，云分类从个人趣味变成国际协议
      - 2017 年 WMO 发布新版《国际云图》，新增「糙面云 Asperitas」等——正是赏云协会（本书作者创立）民间推动的结果，体现公民科学反哺官方标准
      - 云的光学现象研究属大气光学分支，与彩虹、晕、华的几何光学一脉相承（笛卡儿、杨氏干涉等）
  - 维度5·连接与迁移【推导】
    - → 博物学：分类学思维（族-属-种 + 词根拆解）可直接迁移到植物/鸟类/岩石识别
    - → 摄影：光学现象清单（虹彩云、宝光、22 度晕）是风光摄影的「追光」任务清单
    - → 心理/生活：作者「抬头看天减心理医生开销」的论点，可迁移为「日常微观察对抗焦虑」的正念方法
    - → 公民科学：赏云协会推动糙面云入典，是「业余爱好者影响官方标准」的方法论范例

- ④ 反常识预警 🚨（从书里长出，带案例）
  - 🚨 反常识1：「越常见的云越值得收集」。案例：积云天天见、最易收集，反而是新手最该先「收集」到的——稀缺不等于价值高，观云之乐在「日常中看见」而非追逐罕见【原文｜十种主要云彩类型】
  - 🚨 反常识2：云不是「可有可无的摆设」，而是「天空的表情」。日常直觉把云当背景噪音，作者却主张抬头看云能减压、让无所事事变合理——科学观察本身即疗愈【原文｜原话+序言意旨】
  - 🚨 反常识3：业余爱好者能改写官方科学标准。案例：赏云协会推动「糙面云」被世界气象组织正式录入《世界云图》——民间分类趣味反哺国际气象协议，颠覆「命名权只在专家手里」的错觉【原文｜作者协会渊源+学科史】
